# 卒業論文 要件まとめ

## 基本条件
- **論文要旨**：1ページ以内  
- **本論**：10ページ以上  
- **目次**：章立てを適切に行い、自動生成される形で記載  
- **ファイル形式**：LaTeX（PDF出力）
- 言語：日本語

## 環境条件
- **執筆環境**：WSL（Ubuntu）上でLaTeXを使用  
- **LaTeXクラスファイル**：大学配布のテンプレート（例：INIAD.cls）を利用  
- **ビルド**：`latexmk` などによる自動コンパイル  
- **参照管理**：`thesis.bib` を使用し、BibTeXで文献管理  

## 提出物（想定）
- `thesis.tex`（本文）
- `INIAD.cls`（大学テンプレート）
- `.latexmkrc`（ビルド設定）
- `thesis.bib`（参考文献ファイル）
- 出力PDF（完成論文本体）

## 注意事項
- 表紙・要旨・目次・参考文献・付録を含め、全体で**一貫したページ番号付け**を行う。  
- 各章は `\include{chapters/...}` で分割管理。  
- 目次は `\tableofcontents` により自動生成。  
- 英文要旨（Abstract）が必要な場合は、大学指定形式に従う。
